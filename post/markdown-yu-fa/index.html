无序列表

* 项目一 无序列表 `* + 空格键`
* 项目二
* 项目二的子项目一 无序列表 `TAB + * + 空格键`
* 项目二的子项目二

效果：

  • 项目一 无序列表 * + 空格键
  • 项目二
  • 项目二的子项目一 无序列表 TAB + * + 空格键
  • 项目二的子项目二

有序列表

1. 项目一 有序列表 `数字 + . + 空格键`
2. 项目二 
3. 项目三
1. 项目三的子项目一 有序列表 `TAB + 数字 + . + 空格键`
2. 项目三的子项目二

效果：

  1. 项目一 有序列表 数字 + . + 空格键
  2. 项目二
  3. 项目三
  4. 项目三的子项目一 有序列表 TAB + 数字 + . + 空格键
  5. 项目三的子项目二

  • 任务一 未做任务 - + 空格 + [ ]
  • 任务二 已做任务 - + 空格 + [x]
